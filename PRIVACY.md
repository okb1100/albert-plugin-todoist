# Privacy Policy for Albert Todoist Plugin

Last updated: January 2025

## We do not collect, store, or transmit any of your data.

This plugin operates entirely on your local device. Your Todoist API token is stored locally in Albert's configuration directory and is only used to communicate directly with Todoist's official API. No data passes through any third-party servers.

## What this plugin accesses

- Your Todoist tasks, projects, and user information (fetched directly from Todoist's API)
- Your API token (stored locally on your device only)

## What this plugin does NOT do

- Collect analytics or usage data
- Send data to any server other than api.todoist.com
- Share, sell, or transmit your information to any third party
- Track your behavior in any way

## Open Source

This plugin is open source under the MIT License. You can inspect the code yourself at https://github.com/okb1100/albert-plugin-todoist.

## Contact

For questions about this privacy policy, open an issue on the GitHub repository.
